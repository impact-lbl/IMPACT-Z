module  twsclass
 This class defines the linear transfer map and RF field for the TWS beam line elment. Here, the TWS does not include input and output cells. The TWS is simulated using the superposition of two standing wave structure. (G. A. Loew, et al. SLAC-PUB-2295, 1979.)
